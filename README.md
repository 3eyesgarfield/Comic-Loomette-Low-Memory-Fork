# Comic Loomette (Low-Memory Fork) / 漫画织姬改

A local fork of **Comic Looms** that adds a **Low-Memory mode** for browsing and
batch-downloading manga galleries. This fork does **not** auto-update.

- **Userscript:** [`漫画织姬改.user.js`](./漫画织姬改.user.js)
- **Version:** `4.14.10.lowmem.1`
- **License:** MIT

## Installation / 安装

1. Install a userscript manager such as [Tampermonkey](https://www.tampermonkey.net/)
   or [Violentmonkey](https://violentmonkey.github.io/).
2. Open [`漫画织姬改.user.js`](./漫画织姬改.user.js) and install it.
